COD Status Bot Help Center

Getting Started

Welcome to the COD Status Bot help page. Here's how to get started:

  1. Add the bot to your Discord server using this link.
  2. Set up your EZ-Captcha API key (optional but recommended).
  3. Add your Call of Duty account(s) for monitoring.

Commands

Set your EZ-Captcha API key for faster and more frequent checks.

Usage: /setcaptchaservice

Add a new account to monitor. You'll need to provide your SSO cookie.

Usage: /addaccount

Remove a monitored account from the bot.

Usage: /removeaccount

View a list of all your monitored accounts.

Usage: /listaccounts

Perform an immediate status check on your account(s).

Usage: /checknow

Customize how often the bot checks your account status.

Usage: /setcheckinterval

Obtaining Your SSO Cookie

To add your Call of Duty account to the bot, you'll need to obtain your SSO (Single Sign-On) cookie. Follow these steps:

  1. Go to Activision's website and log in with the account you want to track.
  2. Open the developer console in your browser (usually by pressing F12 or right-clicking and selecting "Inspect").
  3. In the console tab, paste the following code: 
    var cookieValue = document.cookie.match(/ACT_SSO_COOKIE=([^;]+)/)[1]; console.log(cookieValue);
  4. Copy the output value. This is your SSO cookie.

Setting Up EZ-Captcha API Key

Using your own EZ-Captcha API key allows for more frequent account checks:

  1. Visit EZ-Captcha's website to register and obtain an API key.
  2. Use the /setcaptchaservice command to set your personal API key.
  3. You can then use /setcheckinterval to customize your account check frequency.

Troubleshooting

If you encounter any issues:

  • Ensure your SSO cookie is up to date.
  • Check if your EZ-Captcha API key is valid and has sufficient balance.
  • Verify that the bot has the necessary permissions in your Discord server.
  • For persistent issues, please contact our support team or check our GitHub repository for known issues.
